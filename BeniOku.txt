- Tüm sıcaklık değerleri otomatik güncellenir

- Aydınlık tema ile kolay geçiş
- Tercih LocalStorage'da saklanır

## 📖 Kullanım Kılavuzu

### Şehir Arama:
1. Arama kutusuna şehir adını yazın (örn: "İstanbul", "Ankara")
2. "🔍 Ara" butonuna tıklayın veya Enter'a basın
3. Hava durumu bilgileri ve 5 günlük tahmin görüntülenecektir

### Konum Kullanma:
1. 📍 butonuna tıklayın
2. Tarayıcı konum izni isteyecektir
3. "İzin Ver" seçeneğini tıklayın
4. Bulunduğunuz yerin hava durumu otomatik yüklenecektir

### Favori Ekleme:
1. Bir şehri arayın
2. Şehir adının yanındaki ☆ ikonuna tıklayın
3. Şehir favorilere eklenecektir (⭐ olarak gösterilir)
4. Favoriler bölümünden hızlıca erişebilirsiniz

### Sıcaklık Birimi Değiştirme:
1. Sağ üstteki °C veya °F butonuna tıklayın
2. Tüm sıcaklık değerleri otomatik dönüştürülür

### Tema Değiştirme:
1. Sağ üstteki ☀️ (aydınlık) veya 🌙 (karanlık) butonuna tıklayın
2. Tema anında değişecektir
3. Tercihiniz kaydedilir

## 🔑 API Bilgileri

### OpenWeatherMap API
- **Mevcut Hava Durumu:** `api.openweathermap.org/data/2.5/weather`
- **5 Günlük Tahmin:** `api.openweathermap.org/data/2.5/forecast`
- **Dil:** Türkçe (lang=tr)
- **Birim:** Metric (Celsius)

### Kendi API Anahtarınızı Kullanın:
1. https://openweathermap.org adresinde ücretsiz hesap oluşturun
2. API anahtarı alın
3. `src/App.jsx` dosyasında `apiKey` değişkenini güncelleyin

```javascript
const apiKey = 'BURAYA_KENDI_API_ANAHTARINIZI_YAPISTIIRIN';
```
