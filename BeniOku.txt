Projeyi yerel makinenizde çalıştırmak için aşağıdaki adımları izleyebilirsiniz:

Depoyu klonlayın:

git clone [https://github.com/SENIN-KULLANICI-ADIN/react-hava-durumu.git](https://github.com/SENIN-KULLANICI-ADIN/react-hava-durumu.git)

Proje dizinine gidin:

cd react-hava-durumu

Gerekli paketleri yükleyin:

npm install

src/App.jsx dosyasını açın ve apiKey değişkenine kendi OpenWeatherMap API anahtarınızı atayın.

Uygulamayı geliştirme modunda başlatın:

npm run dev


Bu proje Ramazan Solak tarafından oluşturulmuştur...
